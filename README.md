# NppThemes

NppThemes is a planned appearance and workspace modernization layer for Notepad++. The project will combine a native plugin, a semantic theme engine, curated theme assets, and safe workspace profiles to deliver a modern, customizable experience without depending on Notepad++ private APIs.

The project is currently in planning and feasibility-validation stage. No installable release exists yet.

## Product direction

- Live theme preview and semantic palette editing
- Complete light and dark editor themes with per-language overrides
- A keyboard-first Theme Studio and command palette
- Reversible focus, minimal, and presentation workspace profiles
- Safe configuration backups and one-action rollback
- Native dark-mode, high-DPI, and accessibility support
- Packages for Notepad++ Plugins Admin after compatibility hardening

See [DEVELOPMENT_PLAN.md](DEVELOPMENT_PLAN.md) for scope, architecture, milestones, quality gates, risks, and release strategy.

## Project status

Repository foundation and development plan are complete. Next step is Phase 0 feasibility prototyping against current Notepad++ public plugin APIs.
