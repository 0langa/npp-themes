# Changelog

All notable changes use [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) structure. Project uses Semantic Versioning.

## [Unreleased]

### Added

- Shared semantic `ShellPalette` derivation with contrast validation and deterministic tests for every built-in profile.
- Opt-in, capability-checked DWM title-bar, caption-text, and border theming with transactional restore, High Contrast precedence, Theme Studio status, and settings persistence.
- Complete plugin-plus-maintained-shell development roadmap for full application-owned window theming.

## [0.1.0] - 2026-07-15

### Added

- Native Notepad++ plugin ABI and dark-mode-aware dockable Theme Studio.
- Six built-in semantic appearance profiles.
- Live dual-view styling and common-language semantic mappings.
- Searchable command palette, theme switcher, and dual-view document navigator.
- Contrast checks and font controls.
- JSON theme import/export with schema and size validation.
- Native Notepad++ theme XML generation from current host model files.
- Focus and Minimal workspace profiles with exact state restoration.
- Presentation workspace with temporary larger editor typography.
- Startup recovery after interrupted workspace sessions.
- Transactional settings, recovery backups, and local diagnostics.
- x64/x86 strict-warning builds, unit tests, host-adapter tests, settings tests, and portable host smoke test.
